# ApplePy-CLI
Python packet sniffer and analyzer built with Click.

# Installation

```{bash}
git clone https://github.com/Matixx22/ApplePy-CLI.git
cd ApplePy-CLI
pip install -r requirenments.txt
pip install --editable .
applepy --help
```

Recommended installing in virtual environment.
